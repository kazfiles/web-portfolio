# Web Portfolio

## 概要
転職活動用に制作したポートフォリオサイトです。  
GSAP（無料版）とScrollTriggerを使用し、
スクロールに応じたアニメーションを実装しています。

## 使用技術
- HTML
- CSS
- JavaScript
- GSAP（ScrollTrigger）

## 制作背景
未経験からWeb制作エンジニアへの転職を目指し、
JavaScriptおよびアニメーションライブラリの理解を深める目的で制作しました。

## 工夫した点
- 見出し・画像にのみアニメーションを適用し、
  情報の視認性を損なわないよう設計しました。
- ScrollTriggerの発火位置を調整し、
  自然な表示タイミングを意識しています。

  ## 今後の改善点
- レスポンシブ対応の強化
- セクション単位でのアニメーション制御の最適化
